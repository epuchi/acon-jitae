1.
사용자와 컴퓨터가 가위바위보하는 프로그램을 작성하려고 한다. 
가위,바위,보는 1,2,3의 숫자로 나타냅니다.
컴퓨터는 랜덤으로 손을 선택합니다. 결과화면에는 ‘무승부’, ‘사용자승리’ ,‘사용자패배’ 중 하나를 표시합니다. 무승부의 경우는 다시 승부합니다.
 (승패가 나면 사용자의  승리 또는 패배를 출력한다 )

  1. 사용자에게 1,2,3 (가위, 바위, 보) 중 입력 받게 한다.
  2. 컴퓨터는 1,2,3 (가위, 바위, 보) 중 랜덤으로 출력하게 한다.
  3. 사용자의 입력과 컴퓨터의 출력을 비교 한다.
  3-1. 사용자의 입력과 컴퓨터의 출력 값이 같다면 '무승부'를 출력하고 끝낸다.
  3-2. 사용자의 입력이 1이고, 컴퓨터의 출력 값이 2이면 '사용자패배'를 3이면 '사용자승리'를 출력하고 끝낸다.
  3-3. 사용자의 입력이 2이고, 컴퓨터의 출력 값이 1이면 '사용자승리'를 3이면 '사용자패배'를 출력하고 끝낸다.
  3-4. 사용자의 입력이 3이고, 컴퓨터의 출력 값이 1이면 '사용자패배'를 2이면 '사용자승리'를 출력하고 끝낸다.


 

2. 3리터짜리 양동이와 5리터짜리 양동이를 하나씩 들고 강에 가서, 딱 4리터의 물을 길어 오려면 어떻게 하면 좋을까요?
  - 다른도구는 없습니다.

  1. 3L 양동이에 물을 가득 받는다.                      (3L 양동이 : 3L, 5L 양동이 : 0L)
  2. 3L 양동이의 물을 5L 양동이에 담는다.                (3L 양동이 : 0L, 5L 양동이 : 3L)
  3. 3L 양동이에 물을 가득 받는다.                       (3L 양동이 : 3L, 5L 양동이 : 3L)
  4. 3L 양동이의 물을 5L 양동이에 가득 찰 때까지 담는다.  (3L 양동이 : 1L, 5L 양동이 : 5L)
  5. 5L 양동이의 물을 버린다.                           (3L 양동이 : 1L, 5L 양동이 : 0L)
  6. 3L 양동이의 물을 5L 양동이에 담는다.                (3L 양동이 : 0L, 5L 양동이 : 1L)
  7. 3L 양동이에 물을 가득 담는다.                       (3L 양동이 : 3L, 5L 양동이 : 1L)
  8. 3L 양동이의 물을 5L 양동이에 담는다.                 (3L 양동이 : 3L, 5L 양동이 : 4L)



3. 수 맞추기 게임
   컴퓨터가 임의의 수를 내면 사람이 그 수를 맞추는 게임이다 . 맞출 때 까지 반복합니다
   (컴퓨터는 100이내의 임의의 수를 낸다)
   up & down으로  컴퓨터가 낸 수를 맞추기 

 
(   추가옵션사항 : 3번 안에 맞추는 맛있는 간식이 제공된다. ( 간식을 받는 사람에게는 적절한 메시지를 출력한다) 
 
 1. 컴퓨터는 1~100 사이의 무작위 수 하나를 정답으로 저장한다.
 2. 카운트라는 변수를 1으로 지정 한다.
 3. 사용자에게 입력을 수를 입력 받는다.
 4. 사용자의 입력 값이 정답 값이 같다면 9번으으로 이동한다.
 5. 사용자의 입력 값이 정답 값보다 크다면 'down'을 출력한다.
 6. 사용자의 입력 값이 정답 값보다 작다면 'up'을 출력한다.
 7. 카운트 값을 1 추가한다.
 8. 3으로 이동한다.
 9. 카운트 값이 3 이하이면 '3번 안에 정답을 맞추셨습니다! 간식 받아가세요!'를 출력 한다.
 10. 종료 한다.


4 .  수맞추기 게임  ver2   
 컴퓨터가 랜덤수를 정합니다. 
 컴퓨터가 랜덤수를 맞추는 게임입니다.
 세 개의 변수가  랜덤수를 맞춥니다. 맞출 때 까지 반복합니다
 세 개의 변수 중에  어떤 변수가  맞췄는지를 사용자가 선택하는 게임입니다.

 1. 컴퓨터는 1~100 사이의 무작위 수 하나를 정답으로 저장한다.
 2. 컴퓨터는 1~100 사이의 무작위 수 3개를 각각 num1, num2, num3 변수에 저장한다.
 3. num1, num2, num3 중 변수가 정답의 값과 같은 변수가 없다면 2번으로 이동한다.
 4. 사용자는 1,2,3(num1, num2, num3) 중 하나를 선택해 입력 받는다.
 5. 사용자가 선택한 변수와 정답의 값이 같다면 '정답'을 출력하고 종료한다.
 6. '실패'를 출력하고 종료한다.


5. 자신의 나이구하기

 1. 사용자는 생년월일(0000-00-00)을 입력 받는다.
 2. 오늘의 날짜를 today 변수에 저장한다.
 3. 나이 변수에 today의 연도에 사용자의 생년월일 연도를 뺀 값을 넣는다.
 4. 사용자의 생년월일의 월일이 today의 월일보다 이후 이면 나이 변수에 -1을 한다.
 5. 나이를 출력한다.
